## What is Vlume?

macOS caps how loud your speakers can go, which leaves a lot of Macs frustratingly quiet for movies, calls and music. Vlume lifts that ceiling - boosting system audio up to **800% (8x)** - while an always-on **limiter** tames peaks so it gets dramatically louder without distorting.

It is built on Apple's modern Core Audio process-tap API, so unlike older boosters there is **no kernel extension, no restart and no admin password**. It lives in the menu bar, remembers your settings, and is a one-time purchase rather than a subscription.

## How it works

1. **Capture** - Vlume taps system audio with Apple's process-tap API and mutes the original.
2. **Boost** - the signal is amplified, protected by the limiter and shaped by your equaliser.
3. **Play** - clean, boosted audio is sent straight to your selected speakers or headphones.
